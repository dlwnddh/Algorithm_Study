1. 숫자 반영 정렬 비교하기

숫자와 문자가 섞인 두 문자열이 주어진다
예시) code10, code3

문자열 비교 후에 오름 차순으로 정렬 했을 때 다를 시 숫자를 반영 안하면
code 10이 code3보다 높은 정렬이 된다.
하지만 문자열안에 숫자를 비교했을 때에는 code3이 우선순위가 높다.

따라서 아래와 같은 법칙으로 정렬한다고 할때 두 문자열 중 어떤 것이 우선순위가 높은지 출력하라

1. 한글자씩 비교하다가 다르면 알고리즘 start
2. 만약 두 문자열 위치 다 숫자이면 이어지는 숫자를 반영하여 최종 숫자를 완성하여 비교
3. 2번이 아니라면 아스키 코드 값으로 비교

3. 이상한 계산기

해당 계산기에는 4개의 버튼이 있다.
1. 0으로 초기화 버튼
2. 주어진 값 +1 버튼(단, 숫자가 999,999라면 0이 된다)
3. 주어진 값 -1 버튼(단, 숫자가 0이라면 999,999가 된다)
4. 주어진 값 *2 버튼 (단, 숫자가 500,000이상이면 6자리 수만 취급한다.)

처음 계산기에 찍힌 값은 0이고
이후 정하고 싶은 값으로 가고 싶을 때 최소한의 버튼으로 가는 횟수를 출력하라
ex) 원하는 숫자 = 7
0->1->2->4->8->7 총 5번
